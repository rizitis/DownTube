# DownTube
*yt-dlp DownLoad Helper*

## Usage
From terminal: `node DownTube.mjs` <br>


## Installation 
If you want to install it and run DownTube from your DE menu, run: `sudo bash install.sh`

## Uninstall
`sudo bash UNINSTALL-DOWNTUBE`
